Drug Info Extractor + Knowledge Graph
Overview

AI system that extracts drug-related information from TXT, DOCX, and PDF files using NLP and builds a knowledge graph using Neo4j.

Features:
Multi-format file processing (TXT, PDF, DOCX)
NLP-based extraction of:
Drug name
Side effects
OTC availability
CSV generation
Interactive UI using Streamlit
Knowledge graph using Neo4j

Tech Stack:
Python
NLP (spaCy)
Streamlit
Neo4j

How to Run:
pip install -r requirements.txt
streamlit run app.py

Workflow:
Files → NLP → CSV → UI → Knowledge Graph
